Gestionale Biblioteca

Una semplice App che si collega ad un server di JSON per inserire i libri con i dati relativi sull server, per poi prenderli e metterli nell'apposita tabella sull'App, dalla quale si possa modificare i dati di ogni libro o eliminare i libri.

* Essendo un apagina pubblica ho scelto di mettere un video dell'app per non esporre le credenziali per il collegamento al server che sono presenti nel codice.
